# 数据库设计说明书

> **项目名称：** 综合物流管理系统
> **文档版本：** V1.0
> **编写日期：** 2026-04-18
> **数据库系统：** PostgreSQL 16

---

## 1. 设计原则

### 1.1 命名规范

| 对象 | 规范 | 示例 |
|------|------|------|
| 表名 | 小写下划线，前缀表模块 | `sys_user`, `wh_location`, `o_order` |
| 字段名 | 小写下划线 | `user_id`, `created_at` |
| 主键 | 统一命名 `id` | `BIGSERIAL PRIMARY KEY` |
| 外键 | `xxx_id` | `warehouse_id`, `order_id` |
| 索引名 | `idx_表名_字段` | `idx_order_status` |
| 唯一约束 | `uk_表名_字段` | `uk_user_username` |

### 1.2 通用审计字段

所有表均包含：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `created_at` | TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | 更新时间 |
| `created_by` | VARCHAR(64) | 创建人 |
| `updated_by` | VARCHAR(64) | 更新人 |

## 5. 外键约束说明

所有外键均设置 `ON DELETE RESTRICT`，防止级联删除导致数据丢失。业务数据删除需先清理关联记录。
