## 2. 通用请求/响应

### 2.1 统一响应

```json
{
  "code": 200,
  "message": "success",
  "data": { ... },
  "timestamp": "2026-04-18T21:00:00Z"
}
```

### 2.2 分页请求

```
GET /api/xxx?page=1&size=10&sort=createdAt,desc
```

### 2.3 分页响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "records": [...],
    "total": 100,
    "page": 1,
    "size": 10,
    "pages": 10
  }
}
```

### 2.4 错误响应

```json
{
  "code": 400,
  "message": "参数校验失败",
  "errors": [
    { "field": "username", "message": "用户名不能为空" }
  ],
  "timestamp": "2026-04-18T21:00:00Z"
}
```

## 3. 接口详细设计（示例）

### 3.1 登录

```
POST /api/auth/login
Content-Type: application/json

Request:
{
  "username": "admin",
  "password": "admin123"
}

Response:
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "expiresIn": 7200,
    "user": {
      "id": 1,
      "username": "admin",
      "displayName": "管理员",
      "roles": ["ADMIN"]
    }
  }
}
```

### 3.2 创建订单

```
POST /api/order/orders
Authorization: Bearer <token>
Content-Type: application/json

Request:
{
  "senderName": "张三",
  "senderPhone": "13800138000",
  "senderAddress": "上海市浦东新区xxx",
  "receiverName": "李四",
  "receiverPhone": "13900139000",
  "receiverAddress": "北京市朝阳区xxx",
  "items": [
    {
      "skuName": "电子元件A",
      "skuCode": "SKU-001",
      "quantity": 100,
      "weightKg": 5.5,
      "volumeCbm": 0.02,
      "unitPrice": 10.00
    }
  ],
  "remark": "轻拿轻放"
}

Response:
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "orderNo": "ORD202604180001",
    "status": 10,
    "statusName": "待确认",
    "totalAmount": 1000.00
  }
}
```

## 4. 订单状态流转

```
10(待确认) → 20(已确认) → 30(已入库) → 40(已发货) → 50(运输中) → 60(已送达) → 70(已完成)
                                              ↓
                                         80(已取消) ← ← ← ← ← ← ←
```

---

## 5. 运单状态流转

```
1(待提货) → 2(配送中) → 3(已送达)
              ↓
            4(拒收)
```
