4. **Leaflet 地图** — `TrackingPage` 为演示模式，坐标基于城市名映射；真实环境需接入地理编码 API

## 五、启动方式（快速参考）

```bash
# 1. Docker 基础设施
cd docker && docker compose up -d

# 2. 后端
cd backend && mvn clean package -DskipTests -q
java -Xms768m -Xmx768m -Xss512k -jar target/logistics-backend-1.0.0.jar &

# 3. 前端
cd frontend && npm run dev -- --port 3000
```

**访问**：`http://localhost:3000` → `admin / admin123`
