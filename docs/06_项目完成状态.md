## 五、启动方式

```bash
# 1. Docker 基础设施
cd docker && docker compose up -d

# 2. 后端
cd backend && mvn clean package -DskipTests -q
nohup java -Xms768m -Xmx768m -Xss512k -jar target/logistics-backend-1.0.0.jar > /tmp/backend.log 2>&1 &

# 3. 前端
cd frontend && nohup npx vite --port 3000 > /tmp/vite.log 2>&1 &
```

**访问**：`http://localhost:3000` → `admin / admin123`
